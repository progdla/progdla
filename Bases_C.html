3) Les Variables

Une variable en programmation est une association d’un type à un identificateur (nom) qui est sensé recevoir de données qui sont d’ailleurs modifiables selon les besoins du programmeur.

3 .1) Déclaration de variable en Langage C et C++

En langage C et C++ ,on déclare une variable en donnant premièrement son type ( char ; short ; Int ; long ; float ; double ; long double ; etc.) suivi de son identificateur(nom) ,cet identificateur est attribué par le programmeur. Nous vous conseillons de donner des identificateurs qui ont trait à la ou aux donné(es) manipulée(es).

Syntaxe de Declaration:   < Type >   < Identificateur> ;

Voici quelques exemples de déclaration :
🔸 Exemple 1 : int  x ; cela dit à la machine de réserver un espace mémoire pour une variable de type int (entier moyen) qui a pour identificateur x.
🔸 Exemple 2 :float  y ; cela dit à la machine d’allouer un espace mémoire pour une variable de type float (réel moyen) qui a pour identificateur y.
🔸 Exemple 3: char   Geo ; char Geo, cela dit à la machine de réserver un espace mémoire pour une variable de type char (caractère) qui a pour identificateur Geo.
🔸 Exemple 4 : char  Mon_nom [ ] ; cela dit à la machine d’allouer un espace mémoire pour une variable de type char [ ] (chaine de caractères) qui a pour identificateur Mon_nom.
🔸 Exemple 5 :int  Vect [ ] ; cela dit à la machine de réserver un espace mémoire pour une variable de type int [ ] (vecteur d’entier moyen) qui a pour identificateur Vect.
🔸 Exemple1 :long  Tab [ ] []; cela dit à la machine d’allouer un espace mémoire pour une variable de type long [ ] [ ] (tableau d’entier long) qui a pour identificateur Tab.

REMARQUE :
Pour les vecteurs (tableaux unidimensionnelles), les Tableaux bidimensionnelles et les chaines de caractères, vous devriez impérativement donner la taille désirée.
Exemple :int vect[10] ;float mat[4][3] ;char prénom[15] .cela signifie respectivement que vect ,mat ,prénom peuvent contenir respectivement au maximum 10 entiers,12 réels, une chaine de 15 caractères .

3 .2) Initialisation

Toutefois, vous êtes amenés à initialiser votre variable dès sa déclaration, alors il suffit de mettre égale(=) puis l’attribuer votre valeur initiale .Cela permet à votre variable de ne pas prendre une valeur aléatoire car la case mémoire allouée n’est jamais vide, il contient toujours et toujours une valeur quelconque .Raison pour laquelle l’initialisation est primordiale car sans initialisation, vous avez une forte chance d’avoir une erreur d’implémentation.
◾ Exemple: int Z = 0 ; float Vect [ ] = {0} ; char F = 'g' ; char wilaya= "Boumerdès" ; ces variables sont initialisées respectivement à 0, {0}, g et Boumerdès .

3.3) L’affectation

En programmation affecter une variable consiste à l’attribuer une valeur ou à modifier son contenu en lui donnant une nouvelle valeur ou expression.
🔶 Exemple d’affectation : Supposons que les variables W et M ont été déclarés de même type et initialisées.
{
   W=7 ;
   M=W ; // affectation de la valeur de W dans M ;
}
Nous aurons comme résultat value de M=W=7 ;

Remarque:
◾ L’opérateur d’affectation en C et C++ est = Exemple : x=34, signifie qu’on met 34 dans x.
◾ L’opérateur d’égalité en C et C++ est ==
◾ Exemple : x==34, signifie la valeur contenue dans x équivaut à 34

7) Les Boucles

Une boucle informatique est une structure de contrôle permettant d’exécuter un ensemble d’instructions de façon répétée sous la base d’une condition booléenne (vrai ou faux). Que ce soit en langage C ou C++, on distingue trois boucles de bases qui sont : for  (pour),  while  (tant que) et  do ... while  (répéter).
Une boucle se compose de quatre éléments essentiels :
◾  Un bloc de code qui sera exécuté un certain nombre de fois
◾  Une condition qui doit porter sur une ou des variables, cette ou ces variables s’appelle (ent) variable de boucle.
◾  Une initialisation : le programmeur doit initialiser le ou les variables de boucles.
◾  La modification : il est recommandé d’indiquer à la machine le pas de la boucle. on part de l’incrémentation si le pas est positif ou de la décrémentation dans le cas contraire.
La boucle est jaugée correcte si et seulement si ces quatre éléments essentiaux sont respectés.

Decrouvons dans cette suite les trois principales boucles:

7.1) La Boucle for

Dans nos diffèrent problèmes, on fait recourt à la boucle for lorsqu’on sait d’avance le nombre de fois que le bloc d’instructions sera exécuté .La particularité de la boucle for est le faite que   l’initialisation ,   la condition  et   la modification  se fait à l’intérieur de cette dernière et séparer par des points-virgules à la différence de la boucle while et do…….while .

Exemple :

Dressez un programme C qui addition les 10 premiers entiers naturels en utilisant la boucle for puis affiche la réponse.

SOLUTION

#include<stdio.h>
int main ( )
{
  int i, som=0;
  for ( i=1 ; i <= 10 ; i++ )
    {
    som=som + i ;
   }
  printf ("La somme des 10 premiers entiers est: %d\t", som) ;
  return 0 ;
}

Remarque :
Lorsqu’on a une seule instruction à exécuter dans la boucle, le bloc (les accolades) est facultatif.

Autrement dit, après le for si vous ne mettez pas d’accolades alors c’est l’instruction qui suit immédiatement qui seule sera exécuté dans le compte de for mais les autres non. Cela est valable pour la boucle while, do……while voir la condition if aussi.

7.2) La Boucle do ... while

Tout Comme la boucle for, le do ... while() fait exactement le même travail mais on fait recours à ce dernier dans la situation où nous savons d'avance que notre notre Bloc d'intruction sera exécuter au moins une fois et c'est le nombre total qui nous reste incertain ( Non connu).
L’initialisation de la variable de boucle se fait toujours hors de cette boucle juste avant le do et la modification dans le bloc d’instructions et termine par specifier la condition entre les parenthèses du while suivit d’un point ; (virgule).

Syntaxe de do ... while :
initialisation ;
do  {
     instruction1

       "

7.3) La Boucle while

La boucle while permet de répéter un bloc d’instruction tant qu’une condition est vérifiée (vraie). L’initialisation se fait bien-sûr avant l'entrée de la boucle while, la condition est reste egalement specifiée entre les deux parenthèses venant juste apres le terme while et on echeve par l'apport de la modification dans le bloc juste apres la tout derniere instruction qui sera executée si la condition est vraie. Cette variable de boucle dont on modifie la valeur sera renvoyé dans le while pour être testée de nouveau et ainsi de suite jusqu’à ce que la condition soit fausse et si c'est le cas,la boucle sera brisée automatiquement(arrêt de la boucle).

Pour bien faire voir la différence qu'existe entre ces trois boucles, nous vous proposons l’exemple du do…….while () (répéter) mais reformuler d’une autre manière comme suit :

Etant donné que nous avons defini notre structure,nous seront amené a manipuler cette dernier et il faloir pourvoir cibile les proprietés de cette structure que soit pour la modification ou la recuperation de la valeur de cette propriété. Cet mecanisme est l'accès aux elements de la structure.
